# Banking Web App

The "Banking Web App" allows to create accounts, add deposits and transfer money to other bank accounts.

Finished date of project: 

***

## Tech Stack

- Blazor Server (.NET 6)
- MongoDB (including Atlas)
- Azure Active Directors B2C

***

## Planing project

1. Features
	- createing accounts
	- see balance
	- add balance
	- transfer balance to other accounts

***

## Data Structure

- User
	- Id
	- Surname
	- Forname
	- Bank Account Number
	- Balance
	- E-Mail

- Transaction
	- Id
	- FromUser (User Id)
	- ToUser (User Id)
	- Amount
